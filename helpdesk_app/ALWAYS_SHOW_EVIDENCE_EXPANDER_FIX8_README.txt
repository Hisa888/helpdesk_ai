# 修正8：回答後の「回答の根拠を見る」常時表示対応

## 対応内容

- 通常回答・候補選択後の回答・RAG回答・該当なしのどの問い合わせでも、回答下に必ず「回答の根拠を見る」アコーディオンを表示するように修正しました。
- 回答直後だけでなく、Streamlit の再描画後・画面スクロール後・履歴再表示後にも根拠アコーディオンが消えないよう、回答履歴メッセージに根拠情報を保持するようにしました。
- 候補ボタンを押して表示された回答にも、選択したFAQを根拠として表示するようにしました。
- 「追加情報を記録（任意）」は引き続き回答下に閉じた状態で表示されます。

## 確認ポイント

1. 通常の質問を入力する
2. 回答の下に「回答の根拠を見る」が表示されること
3. 候補ボタンを押して回答を表示する
4. 回答の下に「回答の根拠を見る」が表示されること
5. 画面をスクロールしても「回答の根拠を見る」が消えないこと
6. 「追加情報を記録（任意）」が閉じた状態で表示されること
