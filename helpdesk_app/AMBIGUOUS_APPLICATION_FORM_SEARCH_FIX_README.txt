# あいまいな申請書検索の精度改善

## 対応内容

以下のような自然文の問い合わせでも、FAQ側の「申請書・書式」FAQを優先して採用するようにしました。

- システム導入するための申請書はどれですか？
- システム導入の申請書を教えてください
- 外部システムを入れるときの書式は？
- サイトアクセス許可の申請書はどれですか？
- URL許可の書式を教えてください

## 修正方針

RAGドキュメント側に似た文章がある場合でも、ユーザーが「どの申請書か」を聞いている場合は、
FAQに登録された正式な対応表を優先します。

## 主な修正ファイル

- helpdesk_app/modules/search_runtime.py
- helpdesk_app/modules/faq_answer_flow_runtime.py

## 確認済み

アップロードされた FAQ（faq_operation_template）を元に、以下のヒットを確認しています。

- 「システム導入するための申請書はどれですか？」
  → 「システム導入（外部システム・新規アプリケーション）の申請書はどの書式を使用したらよいですか？」

- 「サイトアクセス許可の申請書はどれですか？」
  → 「サイトアクセス許可の申請書はどの書式を使用したらよいですか？」
