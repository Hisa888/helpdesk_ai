# 候補ボタン直接回答 FIX

## 修正内容
- 「もしかしてこれですか？」の候補ボタンを押した時、再検索ではなく選択FAQの回答をチャット履歴へ直接追加するように修正。
- ボタン押下後に「追加情報を記録」だけが表示され、回答が出ないように見える問題を修正。
- 同じ候補ボタンを2回押しても、同じ回答が重複追加されにくいように制御。
- 候補表示時は「追加情報を記録（任意）」を表示し、候補選択後は回答済みとして追加情報フォームが出続けないように調整。

## 確認手順
1. 質問を入力して「もしかしてこれですか？」候補を表示する。
2. 候補ボタンを押す。
3. 選択したFAQの回答がチャット履歴に表示されることを確認する。
4. もう一度同じ候補を押しても、同じ回答が連続で増えないことを確認する。
