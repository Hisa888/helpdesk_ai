# 候補ボタン・追加情報フォーム 安定化修正

## 修正内容

1. 「もしかしてこれですか？」候補表示時にも、画面下に「追加情報を記録（任意）」フォームを表示するように修正しました。
   - 候補が違う場合に、端末・利用場所・ネットワーク・影響範囲・エラー内容を補足できます。

2. 候補ボタンを押して回答を表示した後も、上にスクロールしたとき候補ボタン欄が消えないように修正しました。
   - 以前は最後のassistantメッセージにだけ候補ボタンを再描画していたため、候補選択後に古い候補欄が消える場合がありました。
   - 今回は候補表示メッセージごとにボタンを再描画します。

3. 同じFAQ候補が複数回出ても Streamlit のボタンkeyが重複しないように修正しました。
   - 履歴番号を含めた一意キーを使います。

## 確認手順

1. `streamlit run app.py` を実行
2. `demo / demo / demo` でログイン
3. 曖昧な質問を入力して「もしかしてこれですか？」を表示
4. 画面下に「追加情報を記録（任意）」が表示されることを確認
5. 候補ボタンを押す
6. 回答が表示された後、上にスクロールして候補ボタン欄が残っていることを確認
