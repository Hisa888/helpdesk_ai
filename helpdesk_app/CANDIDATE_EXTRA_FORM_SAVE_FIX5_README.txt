候補表示時の追加情報フォーム 修正5

対応内容:
1. 「追加情報を記録（任意）」は初期状態でたたんで表示するように修正。
2. 候補表示・候補選択後に timestamp が空でも保存できるように修正。
3. 「この内容で記録」ボタン押下時、保存失敗にならず nohit_YYYYMMDD.csv に追記/更新できるように修正。
4. 既存機能は削除していません。

確認方法:
- 「ポケットWIFIが使えなくなった。」で候補を表示
- 回答の根拠欄の下に「追加情報を記録（任意）」が閉じた状態で表示されること
- 開いて入力後、「この内容で記録」を押すと保存成功すること
- 候補ボタン押下後も回答欄と追加情報フォームが表示されること
