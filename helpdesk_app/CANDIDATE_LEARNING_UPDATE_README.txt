# もしかして候補クリック学習アップデート

## 追加内容
- 「もしかしてこれですか？」候補ボタンを押した時に、元の質問と選択FAQを学習ログとして保存します。
- 保存先: logs/candidate_learning_YYYYMMDD.csv
- 同じユーザー語で次回検索された場合、過去に選ばれたFAQへスコア補正をかけます。
- セッション内では即時反映し、CSVログも読み込むため再起動後も直近ログを反映できます。

## 例
1. ユーザー: 「モニター死んだ」
2. 候補: 「ディスプレイが映らない」をクリック
3. 次回以降「モニター死んだ」は「ディスプレイが映らない」が上位に出やすくなります。

## 注意
- 誤クリックを学習しすぎないよう、補正値は最大0.30に制限しています。
- FAQ自体は自動更新しません。ログを見ながら、管理者が「キーワード・言い換え」に正式反映する運用がおすすめです。
