# 社内ドキュメントRAG管理 追加内容

## 追加した画面
管理者エリア内の「📚 社内ドキュメントRAG（PDF / Word / Excel / Wiki）」に、以下の一覧を追加しました。

- ファイル名
- 種類
- 取込日
- 登録者
- チャンク数
- 削除ボタン

## 削除ボタンの動作
対象ファイルを削除すると、以下をまとめて実行します。

1. manifest.json の登録情報から削除
2. chunks.json から対象ファイルのチャンクを削除
3. source_files 配下の元ファイルを削除
4. embeddings.npy を削除して、古いベクトル索引が残らないようにする

削除後は、残ったチャンクに対してTF-IDF検索で継続利用できます。
sentence-transformers のベクトル索引は、次回「この内容でRAGへ反映」時に再生成されます。

## 登録者
RAG反映時の登録者は、ログイン中の管理者IDを保存します。
