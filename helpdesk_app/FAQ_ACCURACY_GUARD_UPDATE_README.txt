# 精度改善版アップデート

## 目的
「ディスプレイが映らない」と入力したときに、VPN資料など別領域のドキュメントが高スコアで採用される誤回答を防ぐための修正です。

## 修正内容
- 画面/ディスプレイ系のユーザー語を正規化
  - 例: 画面黒い / 画面真っ暗 / モニター死んだ / 表示されない
- 検索概念に `display` を追加
- FAQ検索時、質問領域と違うFAQへペナルティを追加
  - ディスプレイ質問にVPN/ログイン/パスワード系FAQが上位化する誤爆を抑制
- ドキュメントRAG採用前に軽量ドメインチェックを追加
  - 質問がディスプレイ系なのに、資料がVPN系なら自動採用しない
- 自動回答しきい値を超えていても、FAQ領域が違う場合は「もしかしてこれ？」側へ落とす
- 根拠バーに「ドキュメント一致度（領域不一致のため不採用）」を表示

## 確認してほしい質問例
- ディスプレイが映らない
- 画面黒い
- 画面真っ暗
- モニター死んだ
- VPNにつながらない
