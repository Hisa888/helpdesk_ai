# FAQ更新日・更新者・備考 追加対応

## 追加したFAQ列

FAQ管理のExcel/CSVで、以下の列に対応しました。

- 操作
- FAQ_ID
- 質問
- 回答
- 意図
- キーワード・言い換え
- カテゴリ
- 表示形式
- 有効
- 更新日
- 更新者
- 備考

## 自動設定ルール

FAQを反映したタイミングで、アプリ側が以下を自動設定します。

- 更新日: 反映時の現在日時
- 更新者: ログイン中の管理者名
- 備考: Excel/CSVの「備考」列の内容を取り込み

Excel側に「更新日」「更新者」が入っていても、反映時の値で上書きします。
これは、誰がいつFAQを反映したかをアプリ側の記録として正しく残すためです。

## 更新者名の設定方法

現在の管理者ログインがパスワードのみの場合、標準では「管理者」として記録します。
名前を変えたい場合は、環境変数または Streamlit secrets に下記を設定してください。

ADMIN_DISPLAY_NAME=佐藤

これにより、FAQ反映時の「更新者」が「佐藤」になります。

## 操作列のルール

| 操作 | 処理 |
|---|---|
| + | 追加。FAQ_IDは使わず新規自動採番 |
| 空白 | 更新。FAQ_IDに一致する既存FAQを上書き |
| - | 削除。FAQ_IDに一致する既存FAQを削除 |

## 変更履歴

FAQ差分反映時に、runtime_data/faq_import_history.csv へ履歴を追記します。

履歴列:

- 日時
- 更新者
- 操作
- FAQ_ID
- 質問
- 結果
- 備考
