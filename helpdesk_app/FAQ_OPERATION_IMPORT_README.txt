FAQ差分アップロード仕様

列:
操作, FAQ_ID, 質問, 回答, 意図, キーワード・言い換え, カテゴリ, 表示形式, 有効, 更新日, 更新者, 備考

操作列:
+     追加
空白  更新（~ は使わない）
-     削除

重要:
+ の追加行は、FAQ_ID欄の値を使わず、現在登録されているFAQの末尾から新しいFAQ_IDを自動採番します。
そのため、過去に出力したFAQ_IDが入った追加用ファイルをアップロードしても、既存FAQ_IDとの衝突で止まりません。

更新は FAQ_ID をキーにして既存FAQを上書きします。
削除は FAQ_ID に一致するFAQを削除します。
操作列が無いファイルは、既存FAQを消さないため標準では「追加のみ」として扱います。

更新日・更新者・備考:
更新日 は、FAQ反映時に現在日時を自動設定します。
更新者 は、ログイン中のログインIDを自動設定します。
備考 は、Excel/CSVの備考列の値を取り込みます。

Excel/CSV上の更新日・更新者は、FAQ反映時にアプリ側の値で上書きします。
ログインIDは ADMIN_LOGIN_ID で設定できます。複数管理者は ADMIN_USERS=sato:password1,tanaka:password2 形式に対応します。

FAQ反映履歴:
runtime_data/faq_import_history.csv に、日時・更新者・操作・FAQ_ID・質問・結果・備考を追記します。
