# 問い合わせスレッド管理版

## 追加内容

- 1問い合わせ＝1スレッドIDで管理
- サイドバーに「問い合わせスレッド」を追加
- 「＋ 新規問い合わせ」ボタンを追加
- 未対応 / 対応中 / 解決済みで分類表示
- 会社IDごとに保存先を分離

## 保存先

runtime_data/tenants/<company_id>/threads/inquiry_threads.json

## 使い方

1. サイドバーの「＋ 新規問い合わせ」を押す
2. 通常どおり問い合わせを入力する
3. 問い合わせIDが画面上部のアコーディオンに表示される
4. 管理者はステータスや管理者メモを保存できる

## 注意

既存のFAQ検索、RAG、ログ、ログイン切替機能は削除していません。
まずは画面上で問い合わせ単位に分けるための土台として追加しています。
