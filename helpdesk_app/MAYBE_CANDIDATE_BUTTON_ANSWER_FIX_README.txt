# もしかしてこれですか？候補ボタン回答表示 修正

## 修正内容

「もしかしてこれですか？」の候補ボタンを押しても回答が出ない問題を修正しました。

## 原因

Streamlitはボタン押下時に画面を再実行しますが、候補ボタンが現在の回答描画時だけに表示され、チャット履歴の再描画時には候補ボタン情報が残っていませんでした。
そのため、ボタンイベントを安定して受け取れず、押しても回答につながらない状態になることがありました。

## 対応

- 候補FAQをチャット履歴にも保存
- 履歴再描画時にも候補ボタンを表示
- 候補クリック時は再検索ではなく、選択されたFAQの回答を直接表示
- しきい値や曖昧判定に戻されて再び候補表示になる問題を回避
- 候補クリックログは既存どおり保存
