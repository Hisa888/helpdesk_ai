# もしかしてこれ？候補表示アップデート

## 変更内容
- 曖昧な質問で自動回答しきい値に届かない場合、上位FAQ候補を最大3件表示します。
- 「もしかしてこれですか？」ボタンを追加しました。
- ボタンを押すと該当FAQの質問文で再検索し、回答に進みます。
- 低スコアでも完全な該当なしにせず、一定以上の候補があれば候補表示に切り替えます。

## 追加設定（任意）
検索設定に以下のキーがある場合は利用します。未設定でもデフォルトで動作します。
- maybe_candidate_threshold: 0.06
- maybe_candidate_count: 3
