# 「追加情報を記録」フォーム表示修正

## 修正内容

「もしかしてこれですか？」候補表示時に、画面上では

    候補が違う場合は、下の『追加情報を記録』から状況を補足してください。

と案内しているにもかかわらず、実際には「追加情報を記録」フォームが表示されない問題を修正しました。

## 修正後の動き

候補表示時、候補ボタンの下に次のフォームが表示されます。

    追加情報を記録（任意）

フォームから以下を記録できます。

- 端末
- 利用場所
- ネットワーク
- 影響範囲
- エラー内容

## 対象ケース

- FAQ候補はあるが一致度が低い場合
- 「もしかしてこれですか？」の候補が表示された場合
- 候補が違うため、ユーザーが状況を補足したい場合

## 修正ファイル

- helpdesk_app/modules/query_flow_runtime.py
- helpdesk_app/modules/main_screen_layout.py
- helpdesk_app/modules/answer_panel.py

## 備考

候補表示時も nohit ログを作成し、補足情報を同じログ行へ追記できるようにしています。
