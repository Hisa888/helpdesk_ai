# FAQとかけ離れた質問の候補表示抑制 修正

## 対応内容

「今日はしんどい」など、情シスFAQとは関係が薄い入力で、低スコアのFAQ候補が3件表示される問題を修正しました。

## 変更点

- `helpdesk_app/modules/query_flow_runtime.py` に業務ドメイン判定を追加
- Wi-Fi / PC / VPN / 申請書 / 書式 / エラーなどの情シス系キーワードがない場合
- かつ FAQ一致度が低い場合
- 「もしかしてこれですか？」候補を出さずに「該当なし」扱いにします

## 調整値

検索設定に以下を追加すると、しきい値を調整できます。

```python
out_of_domain_candidate_threshold = 0.12
```

値を上げるほど、雑談・FAQ外の入力で候補が出にくくなります。

## 例

- `今日はしんどい` → 候補を出さない
- `眠い` → 候補を出さない
- `ポケットWIFIが使えなくなった` → Wi-Fi系なので候補表示対象
- `システム導入の申請書はありますか` → 申請書・システム系なので候補表示対象
