# PCロック系あいまい検索・誤回答防止修正

## 修正目的
「パソコンがロックされました」のような質問で、
「PCの起動が遅い」「スタートアップアプリが原因」などの無関係なFAQを自動回答しないようにしました。

## 修正内容
- `ロック / ロックされた / アカウントロック / ロック解除` を検索トークンとして強化
- ロック系の質問では、ロックを含まないFAQを強く減点
- ロック系のFAQを候補プールへ必ず入れる
- 「PC + ロック」は意味が複数あるため、自動回答せず「もしかしてこれですか？」へ回す
- 候補には「アカウントがロックされました」などのロック系FAQが出やすくなる

## 期待動作
- パソコンがロックされました
  - すぐ誤回答しない
  - 「もしかしてこれですか？」としてアカウントロック系候補を表示
- アカウントがロックされました
  - アカウントロックFAQへヒット
- パソコンが重いです
  - PC本体/起動遅延系FAQへヒット

## 補足
「パソコンがロック」は、画面ロック・Windowsログイン・AD/M365アカウントロック・BitLockerなど意味が分かれるため、
本番運用では候補表示または追加質問に回す方が安全です。
