# 申請書・書式検索精度改善

## 修正内容

「システム導入するための申請書はどれですか？」のような自然文で、
FAQの「システム導入（外部システム・新規アプリケーション）の申請書」にヒットしやすくしました。

主な改善:
- 「するための」「どれですか」など検索ノイズを正規化
- 「申請書」「書式」「システム導入」「外部システム」「新規アプリケーション」を検索トークン化
- 申請書/書式系FAQの候補プール補強
- システム導入FAQを加点し、トライアル用FAQは質問にトライアル語が無い場合は減点

## 確認した質問例

- システム導入するための申請書はどれですか？
- システム導入の申請書を教えてください
- 外部システムを導入したい場合の書式は？

期待される回答:
システム作業申請書「書式3_システム作業申請書_責任者承認まで.xlsx」でお願いします。
