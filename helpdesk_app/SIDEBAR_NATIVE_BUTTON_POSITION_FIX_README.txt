# サイドバー開閉ボタン位置調整 修正メモ

## 修正目的
左上の Streamlit 標準「<< / >>」で管理メニューを閉じたあと、
「>>」ボタンが画面中央寄りに表示される問題を修正しました。

## 修正方針
独自の「>>」ボタンは追加せず、Streamlit 標準ボタンだけを使用します。
そのうえで、閉じた状態を JavaScript で検知し、標準の「>>」を左上に固定します。

## 修正内容
- `modules/app_bootstrap.py` を修正
- サイドバー閉鎖状態を検知する `data-oai-sidebar-collapsed` を追加
- 閉鎖中は青いリサイズバーを非表示
- 閉鎖中は標準「>>」ボタンを `left: 16px / top: 16px` に固定
- 再表示後はリサイズバーをサイドバー右端に戻す
- 独自ボタンとの競合は発生しないよう、独自ボタンは作成しない

## 確認ポイント
1. 左上の「<<」で管理メニューを閉じる
2. 「>>」が画面左上に1つだけ表示される
3. 「>>」を押すと管理メニューが再表示される
4. 青いリサイズバーは閉じた状態では非表示、開いた状態ではサイドバー右端に表示される
