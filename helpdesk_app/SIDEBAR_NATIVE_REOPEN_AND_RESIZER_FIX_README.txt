# サイドバー開閉ボタン・青い縦バー修正

## 修正対象
- helpdesk_app/modules/app_bootstrap.py

## 修正内容
- 独自の「>>」ボタン追加方式は使わず、Streamlit標準の「<< / >>」だけで開閉する方針に戻しました。
- サイドバーを閉じた状態では、カスタムの青い縦リサイズバーを完全に非表示にします。
- サイドバーを「>>」で再表示した後は、青い縦リサイズバーを管理メニュー右端に再配置します。
- Streamlitのバージョン差異に対応するため、`aria-expanded` と実際の表示位置の両方でサイドバーの開閉状態を判定します。
- MutationObserverでStreamlit標準ボタンの開閉状態変化を監視し、閉じた後・開いた後の表示ずれを補正します。

## 確認ポイント
1. 初期表示で左メニューが表示されること。
2. 左上の「<<」で左メニューが閉じること。
3. 閉じた状態で青い縦バーが中央付近に残らないこと。
4. 左上のStreamlit標準「>>」が残ること。
5. 「>>」を押すと左メニューが再表示されること。
6. 再表示後、青い縦バーが左メニュー右端に戻ること。
7. 「>>」ボタンが二重表示されないこと。
