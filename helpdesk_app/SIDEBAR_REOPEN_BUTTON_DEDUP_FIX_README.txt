# サイドバー復帰ボタン二重表示 修正

## 修正目的
Streamlit標準の「>>」ボタンと、アプリ独自の「>>」ボタンが同時に表示される問題を修正しました。

## 修正内容
- 折りたたみ時は、Streamlit標準の復帰ボタンを非表示化
- アプリ独自の「>>」ボタンだけを表示
- 独自「>>」ボタンから内部的にStreamlit標準の展開処理を呼び出す設計は維持
- 左余白・青い縦バーが残らない前回修正も維持

## 対象ファイル
- helpdesk_app/modules/app_bootstrap.py

## 確認方法
1. `streamlit run app.py` で起動
2. 左上の「<<」で管理メニューを閉じる
3. 「>>」が1つだけ表示されることを確認
4. 表示された「>>」を押すと管理メニューが再表示されることを確認
