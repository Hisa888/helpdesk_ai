# サイドバー再表示ボタン修正

## 修正目的
Streamlit左上の「<<」でサイドバーを閉じたあと、再表示用の「>>」ボタンが消える問題を修正しました。

## 修正内容
- 折りたたみ時だけ表示される独自の「>>」ボタンを追加
- 「>>」クリック時にStreamlit標準のサイドバー展開処理を呼び出し
- 既存のサイドバー幅ドラッグ機能は維持
- 展開後は独自の「>>」ボタンを自動非表示
- 左側の余白と青いドラッグバーが残らないようにした前回修正も維持

## 修正ファイル
- helpdesk_app/modules/app_bootstrap.py

## 確認ポイント
1. 通常表示で左メニューが表示される
2. 左上の「<<」を押す
3. 左メニューが閉じる
4. 画面左上付近に「>>」が表示される
5. 「>>」を押すと左メニューが再表示される
6. 青い縦バーだけが残らない
