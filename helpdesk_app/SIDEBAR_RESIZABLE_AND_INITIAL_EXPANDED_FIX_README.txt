# サイドバー幅調整・初期表示修正

## 修正日
2026-05-04

## 対応内容

1. 左メニューと中央画面の境界をマウスドラッグで幅調整できるように修正
   - 青い縦バーを左メニューの右端に配置
   - 左右ドラッグで左メニュー幅を変更
   - ダブルクリックで初期幅に戻す
   - 幅はブラウザ localStorage に保存され、再読み込み後も反映

2. 初期画面で左メニューを表示するように修正
   - `st.set_page_config(..., initial_sidebar_state="expanded")` を指定

3. 安定性対策
   - 独自の「>>」ボタンは作成しない
   - Streamlit 標準の「<< / >>」で開閉
   - MutationObserver / setInterval による監視は使用しない
   - 青い縦バーは `stSidebar` の子要素として配置し、左メニューを閉じると一緒に非表示

## 修正ファイル

- `helpdesk_app/modules/app_bootstrap.py`

## 確認ポイント

- 初回表示時に左メニューが開いていること
- 左メニュー右端の青い縦バーをドラッグして幅変更できること
- 「<<」で左メニューを閉じられること
- 「>>」で左メニューを再表示できること
- 閉じた状態で青い縦バーが中央に残らないこと
- 応答停止しないこと
