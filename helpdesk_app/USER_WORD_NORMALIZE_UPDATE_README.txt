# ユーザー語吸収（正規化）版

## 修正内容
- 検索前の正規化を強化しました。
  - モニター / モニタ / 画面 / ディスプレー → ディスプレイ
  - 真っ暗 / 黒い / ブラックアウト / 映らん / 表示されない → 映らない
  - 死んだ / 壊れた / 故障 / 反応しない → 映らない 不具合
  - つかない / 付かない / 点かない → 入らない
- display概念を追加し、ディスプレイ系FAQを候補に残しやすくしました。
- 「モニター死んだ」「画面黒い」のような短いユーザー語では、追加質問より「もしかしてこれ？」候補を優先するようにしました。
- VPNなど別領域のRAG誤採用を抑えるペナルティは維持しています。

## 確認してほしい入力例
- モニター死んだ
- 画面黒い
- ディスプレイ真っ暗
- モニター映らない
