# 音声入力機能 追加内容

Chrome / Edge 推奨のブラウザ音声入力を追加しました。

## 2026-05-02 クリック不可対策

当初は画面右下に固定マイクボタンを表示していましたが、Streamlit の固定フッター、翻訳ボタン、チャット入力欄などと重なり、環境によってクリックできない場合がありました。

そのため、右下固定ボタンを廃止し、チャット入力欄の直前に通常のボタンとして
「🎤 音声入力する」を表示する方式に変更しました。

## 仕様

- Chrome / Edge 推奨
- 音声入力ボタンを押す
- 問い合わせ内容を話す
- 文字起こし結果が既存のチャット入力欄へ入る
- 自動送信は安全優先で OFF
- 既存のFAQ検索・回答ロジックは変更していません

## 変更ファイル

- 追加/更新: helpdesk_app/modules/voice_input_panel.py
- 更新: helpdesk_app/modules/chat_input_panel.py

## 注意

- ブラウザ側でマイク利用を許可する必要があります。
- 非対応ブラウザでは通常のテキスト入力を利用してください。

## 2026-05-02 追加修正（入力欄内マイク）
- 入力欄の上に表示していた「音声入力する」ボタンを廃止
- Google検索のように、画面下部のテキストボックス内へ🎤ボタンを表示
- 既存のFAQ検索・回答ロジックは変更なし
- 自動送信はOFFのまま（認識後、内容確認して送信）
